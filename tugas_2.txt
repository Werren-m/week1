Teori :

apa itu git ?
    git adalah version control system untuk men-track update yang di lakukan terhadap source code.

untuk apa fungsi dari command :

git init : 
    git init digunakan untuk meng inisialisasikan git repository kosong di local directory.

git remote add origin <link_name>
    menghubungkan local directory ke git repository

git add <file_name>
    memilih file untuk di kirimkan ke git repository

git commit -m "<message_name>"
    mengkonfirmasi item untuk di kirim dan juga membuat deskripsi tentang update yang di lakukan.

git push origin master
    mengupload file ke repository.
